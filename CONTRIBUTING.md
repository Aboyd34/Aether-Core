# Contributing to Aether

Thank you for your interest in contributing. Aether is mission-critical software for people who rely on privacy to stay safe.

## How to Contribute

### 1. Read the Threat Model
Understanding the risks informs every design choice.

### 2. Open an Issue
Use GitHub Issues for:
- Bugs
- Security concerns (non-critical)
- Feature proposals
- Documentation improvements

### 3. Submit a Pull Request
PRs must:
- Be clear and well-documented
- Pass linting and tests
- Include security considerations
- Avoid adding dependencies without discussion

## Coding Standards

- Language: Go  
- Cryptography: Use vetted libraries only  
- No new crypto primitives  
- No logging of sensitive data  

## Community Guidelines

Contributors must adhere to the Code of Conduct.
